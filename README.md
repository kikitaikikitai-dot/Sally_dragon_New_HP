# Sally Dragon (サリー・ドラゴン) | 海外宣伝用HPプロジェクト

## 🐉 プロジェクト概要
本プロジェクトは、神域のサラブレッドであり、伝統的な占術・霊視のプロフェッショナルである「サリー・ドラゴン（聖龍）」の海外在住日本人向け公式ホームページです。
イギリス・ドイツ・アメリカなどの海外拠点の常連客、および新しく知的探求を求める層へ向けた「本物の秘匿鑑定」への入り口となります。

### ターゲット層
*   海外在住の日本人（特に欧米圏）
*   経営者、政治、警察、作家など「秘密」を抱えるプロフェッショナル
*   量産型スピリチュアルではなく、歴史と研鑽に裏打ちされた「本物」を求める層
